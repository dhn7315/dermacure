# **App Name**: Ophira AI Skin Health

## Core Features:

- Image Upload and Analysis: Allows users to upload images of their skin for analysis using a pre-trained model.
- AI-Powered Skin Condition Prediction: Uses a pre-trained AI model to predict potential skin conditions based on the uploaded image. Returns probability scores for each predicted condition.
- Personalized Skincare Recommendations: Generates personalized skincare routine recommendations using a tool that determines which course of action is best (natural remedies or conventional treatments) depending on the AI model prediction. 
- 3D Element Showcase: Showcases treatments through attractive 3D elements that are visually engaging.
- GSAP Animations: Use GSAP (GreenSock Animation Platform) for smooth, engaging animations throughout the website to enhance user experience.

## Style Guidelines:

- Primary color: Soft, natural green (#A7C4A1) to evoke health and nature.
- Background color: Very light desaturated green (#F0F4EF) to create a clean and calming backdrop.
- Accent color: Muted yellow (#C9B076) for highlights and interactive elements.
- Body and headline font: 'Alegreya', a humanist serif font, lending an elegant, intellectual feel to headings and body text.
- Use minimalist, line-based icons in the muted yellow (#C9B076) to represent different skin conditions and treatments.
- Clean, modern layout with a focus on whitespace to provide a premium feel.
- Subtle animations on hover and page transitions to enhance interactivity without being distracting. Incorporate GSAP for smooth effects.
